# MedicAssist Project Constitution

## Purpose

MedicAssist is an offline, CPU-first AI-powered Progressive Web App that converts unstructured medical audio and text into structured FHIR R4 JSON for emergency healthcare documentation.

## Core Principles

### Offline First

The core functionality must operate without internet connectivity.

### CPU First

All AI inference must execute on CPU hardware without requiring GPUs or cloud services.

### Privacy First

Patient information is processed locally and must never leave the device unless explicitly exported or synchronized by the user.

### Open Source

The project uses Free and Open Source Software and is released under the GNU AGPL-3.0 License.

### Standards Compliance

Structured output must conform to the FHIR R4 specification whenever possible.

### Code Quality

All contributions must pass formatting, linting, testing, and security checks before merging.
