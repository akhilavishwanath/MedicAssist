# Security Policy

## Reporting a Vulnerability

If you discover a security issue in MedicAssist, please report it privately by contacting the project maintainers instead of creating a public issue.

Please include:

- Description of the issue
- Steps to reproduce
- Expected behavior
- Actual behavior
- Relevant screenshots or logs

## Security Principles

- Offline-first processing
- CPU-only AI inference
- No cloud dependency
- Local data storage
- Patient data is never transmitted without user consent
