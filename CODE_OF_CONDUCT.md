# Code of Conduct

## Our Commitment

We are committed to maintaining a respectful, inclusive, and collaborative environment for everyone.

## Expected Behavior

- Be respectful to all contributors.
- Welcome constructive feedback.
- Collaborate openly and professionally.
- Focus on improving the project.

## Unacceptable Behavior

- Harassment or discrimination
- Offensive language
- Personal attacks
- Sharing confidential information

## Enforcement

Project maintainers may remove contributions or take appropriate action if this Code of Conduct is violated.
